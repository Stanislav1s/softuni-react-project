MyFoody Recipes App

A React application that allows users to browse, save, and manage their favourite recipes with dynamic interactions and user authentication.

Features 
    •   Create, edit, browse recipes.
	•	Recipe Details: View detailed recipe information including ingredients, step-by-step instructions, and nutritional info.
	•	Like System: Like/unlike recipes with real-time updates to backend and UI.

Tech Stack
	•	Frontend: React, JavaScript, Bootstrap 5
	•	State Management: React useState and useEffect
	•	Routing: React Router
	•	Data Storage: JSON Server or custom backend (for recipes and likes)
	•	Styling: CSS Modules

Getting Started
	1.	Clone the repository
    2.  "cd server" -> node server.js
    3.  "cd client" -> npm i 
    4.  npm run dev
